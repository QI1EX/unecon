# SQL
Курс посвящен освоению реляционных баз данных и продвинутых техник написания запросов. В процессе обучения был пройден путь от базовой фильтрации данных до проектирования сложных аналитических отчетов с использованием современных стандартов SQL. Особое внимание уделялось производительности запросов и логике обработки данных в среде PostgreSQ

## Основные инструменты и навыки, полученные при обучении:
- Основы SQL (фильтрация, группировка, сортировка, объединение и пр.)
- Подзапросы и табличные выражения (СТЕ)
